## 3) Explotación
- Estado: en progreso.
- Vector inicial: credenciales por defecto `admin:admin` en panel de login staff → acceso completo al dashboard.
- Sistema identificado: **ZoneMinder 1.37.63** (gestión CCTV open-source).
- CVE identificado: **CVE-2024-51482** — Boolean-based SQL Injection crítico.
  - Afecta versiones `1.37.*` hasta `1.37.64` (nuestra versión está dentro del rango).
  - Requiere autenticación (cualquier usuario de bajo privilegio).
  - Permite ejecutar comandos SQL arbitrarios sobre la base de datos subyacente.
  - Utilidad en este contexto: extracción de hashes de usuarios, credenciales reutilizables, datos internos.
  - **Limitación**: SQLi no es RCE directo. Con acceso admin ya obtenido, el valor está en encontrar credenciales reutilizables (SSH).
- Vector RCE alternativo (admin autenticado): command injection vía configuración de monitor (campos FFmpeg/Source).
- Estado de pruebas actual:
  - PoC de GitHub para CVE-2024-51482 clonado localmente.
  - Validación ejecutada con formato correcto de `-i` (host/IP sin esquema `http://`).
  - Resultado: vulnerabilidad confirmada por el PoC.
  - Extracción de hash de credenciales vía SQLi blind completada.
  - Password crackeada con diccionario: `opensesame`.
- Vulnerabilidad/técnica utilizada: **CVE-2024-51482 confirmada** (Blind SQL Injection autenticada).
- Comandos exactos / PoC:
  - `python3 CVE-2024-51482.py -i cctv.htb -u admin -p admin --test --debug`
